# Vanilla UJS

It is implementation of Rails [jQuery UJS][jq-ujs] in pure JavaScript.
No extra dependencies.

## Install

Use [Bower][bower]:

    bower install vanilla-ujs

## Contribute

1. Clone repo

      $ git clone git://github.com/hauleth/vanilla-js.git
      $ cd vanilla-js/

2. Install dependencies

      $ npm install

3. Run tests

      $ grunt test
